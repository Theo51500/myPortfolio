A propos

Qui suis-je ?

Je suis un développeur front end passionné par la programmation. J'ai commencé à apprendre le HTML CSS et Javascript en autodidacte à l'aide de site comme OpenClassRoom, FreeCodeCamp, ou encore Frontendmentor.io pour m'entrainer sur des projets qui se rapprochent du concret.

J'ai pour le moment acquis les connaissances nécessaires pour la création de site vitrine à l'aide de HTML, CSS et un peu de JavaScript. Je recherche alors aujourd'hui des projets concrets afin de me construire un portfolio professionnel.
